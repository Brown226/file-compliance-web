## 1. 用户权限范围

### 1.1 角色定位

普通用户（USER）是核审通平台的主要使用者，负责：
- 创建和提交文档审查任务
- 查看和管理自己创建的任务
- 查阅企业标准库
- 查看个人数据面板

### 1.2 功能清单

| 功能模块 | 权限 |
|----------|------|
| 监控仪表盘 | 仅个人数据 |
| 新建审查任务 | 有权限 |
| 任务历史 | 仅自己创建的任务 |
| 任务详情 | 仅自己创建的任务 |
| 标准库 | 查阅（不能修改） |
| 审查规则 | 查阅（不能修改） |
| 系统管理 | 无权限 |
| 审计日志 | 无权限 |
| LLM配置 | 无权限 |

## 3. 创建审查任务

### 3.1 选择审查模式

点击左侧「新建」，进入审查模式选择页面。系统提供7种审查模式，覆盖不同审查需求：

| 模式 | 图标 | 适用场景 | 说明 |
|------|:----:|----------|------|
| **全量审查** | — | 全面深度审查 | 执行所有规则和AI审查，覆盖错别字、语法、规范合规、一致性等多维度检查，适合重要文档的完整审查 |
| **以库审文** | 📚 | 标准合规检查 | 使用企业标准库+规则引擎+AI进行合规性检查，自动比对文档内容是否符合企业规范标准 |
| **以文审文** | 🔄 | 参照比对审查 | 上传上游参照文件（如设计输入文件），与待审文件进行比对，检查是否一致或存在偏差 |
| **全文一致性** | 🔗 | 跨文件一致性 | 检查多个文件之间的参数、数据、表述是否一致，适合多专业协同文档的交叉核对 |
| **错别字/语法** | 🔤 | 轻量文本检查 | 专注检查错别字、语法错误、标点问题，处理速度快，适合日常文档快速校对 |
| **多模态识别** | 🖼️ | 图表公式识别 | 针对表格结构化数据、数学公式、工程图纸进行智能识别和分析，适合含复杂图表的文档 |
| **自定义规则** | ⚙️ | 专项检查 | 仅执行管理员预设的自定义规则（如特定术语检查、格式规范等），适合针对性检查 |

#### 模式选择建议

| 您的需求 | 推荐模式 |
|----------|----------|
| 重要文档全面检查 | 全量审查 |
| 检查是否符合公司规范 | 以库审文 |
| 核对文档与输入文件是否一致 | 以文审文 |
| 多份文件数据是否一致 | 全文一致性 |
| 快速校对错别字 | 错别字/语法 |
| 文档含大量表格/公式/图纸 | 多模态识别 |
| 按特定规则检查 | 自定义规则 |

### 3.2 操作步骤

1. 选择审查模式
2. 上传文件（支持多文件，单个不超过100MB）
3. 选择参考标准（库审文模式下）
4. 点击「提交任务」

### 3.3 支持文件格式

| 类型 | 格式 |
|------|------|
| Word文档 | .docx, .doc |
| Excel表格 | .xlsx, .xls |
| PPT演示 | .pptx, .ppt |
| PDF文档 | .pdf |
| 工程图纸 | .dwg |
| 文本文件 | .txt, .md |

### 3.4 任务状态说明

| 状态 | 含义 |
|------|------|
| 待处理 | 任务已提交，等待系统处理 |
| 处理中 | AI正在审查，可实时查看进度 |
| 已完成 | 审查完成，可查看结果 |
| 处理失败 | 审查出错，需重新提交 |

## 4. 查看审查结果

### 4.1 进入任务详情

1. 点击左侧「历史」
2. 找到目标任务
3. 点击「查看详情」

### 4.2 结果页面布局说明

任务详情页面分为三个区域：

```
┌─────────────────────────────────────────────┐
│  任务信息栏（标题、状态、提交时间）            │
├──────────────┬──────────────┬───────────────┤
│   文件列表    │   问题统计    │   原文预览    │
│  （左侧）    │  （中部）    │   （右侧）    │
├──────────────┴──────────────┴───────────────┤
│            问题详情列表                      │
│  ┌─────────────────────────────────────┐   │
│  │ 问题1  [定位] [忽略]                 │   │
│  │ 问题类型 | 原文位置 | 问题描述 | 建议 │   │
│  └─────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

#### 区域功能说明

| 区域 | 功能说明 |
|------|----------|
| **文件列表** | 展示本次任务上传的所有文件，点击切换查看不同文件的审查结果 |
| **问题统计** | 统计各类问题的数量分布，直观了解文档质量概况 |
| **原文预览** | 显示文档原文内容，支持按4000字符分片浏览 |
| **问题详情** | 列出所有发现的问题，每条问题包含类型、位置、描述和建议 |

### 4.3 问题类型说明

| 类型 | 图标 | 含义 | 严重程度 | 处理建议 |
|------|:----:|------|:--------:|----------|
| TYPO | 🔤 | 错别字/拼写错误 | 低 | 直接修改文字 |
| GRAMMAR | 📝 | 语法错误 | 中 | 按建议调整语句 |
| VIOLATION | ❌ | 违反规范 | 高 | 必须整改，对照标准修改 |
| INCONSISTENCY | ⚠️ | 表述不一致 | 中 | 核对后统一表述 |

### 4.4 问题操作

| 操作 | 功能 | 使用场景 |
|------|------|----------|
| **定位** | 跳转原文位置，高亮显示问题所在 | 需要查看问题上下文时 |
| **忽略** | 标记为已知误报，不再显示 | 确认是AI误报时使用 |
| **上一个/下一个** | 快速导航到相邻问题 | 逐条处理问题 |
| **导出报告** | 下载PDF/Word格式的完整报告 | 存档或提交审核 |

#### 定位功能详解

点击「定位」按钮后：
1. 右侧原文预览区域自动切换到对应文件
2. 自动定位到问题所在的文本分片
3. 问题原文以**高亮背景**显示
4. 页面顶部显示问题描述浮层

> **提示**：定位功能依赖原文缓存，如首次定位可能需要短暂加载。

### 4.5 问题详情卡片说明

每条问题以卡片形式展示，包含以下信息：

| 字段 | 说明 |
|------|------|
| **问题类型** | TYPO/GRAMMAR/VIOLATION/INCONSISTENCY |
| **严重程度** | 低/中/高 |
| **原文位置** | 所在文件、段落或页码 |
| **问题描述** | AI识别出的具体问题 |
| **修改建议** | AI提供的修改参考 |
| **操作按钮** | 定位、忽略 |

### 4.6 导出报告

点击「导出报告」，支持两种格式：

| 格式 | 适用场景 | 特点 |
|------|----------|------|
| PDF报告 | 存档、打印、提交审核 | 格式固定，不可编辑，适合正式场合 |
| Word报告 | 进一步编辑、添加批注 | 可编辑，适合团队协作修改 |

报告包含内容：
- 任务基本信息（标题、提交时间、审查模式）
- 问题统计概览（总数、各类型数量）
- 问题详情列表（含原文位置、问题描述、修改建议）
- 文件清单

### 4.7 DWG图纸审查特殊说明

对于工程图纸（.dwg文件），审查结果页面提供：

| 功能 | 说明 |
|------|------|
| **SVG预览** | 矢量渲染显示图纸内容，支持缩放和平移 |
| **错误定位** | 点击问题可高亮对应图元（红色描边+发光效果） |
| **图元信息** | 通过data-handle属性关联审查问题与图纸元素 |

> **操作**：鼠标滚轮缩放图纸，拖拽平移，点击「适应窗口」恢复全图显示。

## 7. 实用技巧

### 7.1 文件命名规范

建议使用规范的命名方式，便于管理和检索：

```
项目简称_文档类型_版本号_日期.扩展名

示例：
田湾7号_技术方案_v1.2_20260424.docx
海阳3号_设计说明_v2.0_20260424.pdf
```

### 7.2 任务提交建议

| 建议 | 说明 |
|------|------|
| 分模式提交 | 不同审查需求创建不同任务 |
| 控制文件数量 | 单次任务建议不超过10个文件 |
| 及时查看 | 处理完成后尽快复核结果 |
| 定期归档 | 完成审查后及时导出报告存档 |

### 7.3 问题处理技巧

1. 优先处理VIOLATION（违反规范）类型问题
2. 对确认无误的误报点击「忽略」
3. 使用「定位」功能快速跳转原文位置
4. 导出报告后按问题列表逐项修改文档

## 8. 常见问题

### Q1：为什么看不到其他同事的任务？

普通用户只能查看自己创建的任务，不能查看他人的任务。如需查看部门任务，请联系部门主管。

### Q2：任务一直处于「处理中」怎么办？

1. 耐心等待，大型文件可能需要较长时间
2. 刷新页面查看状态更新
3. 如超过10分钟仍无变化，联系部门主管或管理员

### Q3：文件上传失败怎么办？

1. 检查文件大小是否超过100MB限制
2. 检查文件格式是否在支持列表中
3. 检查网络连接是否正常
4. 尝试重新上传

### Q4：AI审查结果不准确怎么办？

1. 先确认是否为真正的误报
2. 对误报问题点击「忽略」
3. 如频繁出现同类误报，向部门主管反馈调整审查规则

### Q5：如何修改密码？

1. 点击右上角用户名
2. 选择「修改密码」
3. 输入原密码和新密码
4. 点击确认

### Q6：忘记密码怎么办？

联系部门主管或管理员重置密码。
