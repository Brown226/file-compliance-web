## 1. 项目结构

```
file-compliance-web/
├── backend/                    # Node.js 后端
│   ├── src/                   # TypeScript 源码
│   ├── prisma/               # 数据库 Schema
│   └── markitdown-service/   # Python 文档解析服务
├── frontend/                  # Vue 3 前端
├── docs/                      # 文档
└── MaxKB-2.8.0/              # MaxKB 知识库（可选）
```

## 7. 离线部署关键点

### 7.1 必须包含的镜像

1. **pgvector/pgvector:pg15** - 数据库
2. **redis:7-alpine** - 缓存
3. **markitdown:v1.0** - 文档解析（自定义构建）
4. **backend:v1.0** - 后端 API（自定义构建）
5. **frontend:v1.0** - 前端 Web（自定义构建）

### 7.2 可选镜像（MaxKB）

如需知识库功能，需要额外镜像：
- `maxkb-offline:v2.8.0`
- `maxkb-postgres-offline:v2.8.0`
- `maxkb-redis-offline:v2.8.0`
